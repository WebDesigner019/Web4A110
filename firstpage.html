Judul Materi: Pengantar Pemrograman Web dan Pengenalan Tools Deskripsi Materi: Pemrograman web adalah proses pembuatan aplikasi berbasis web yang dapat diakses melalui browser internet. Dalam pengembangan web, terdapat dua komponen utama: A. Front-End (Client-Side): Bagian dari aplikasi web yang berinteraksi langsung dengan pengguna. Ini mencakup: a) HTML (HyperText Markup Language) → Untuk struktur halaman web b) CSS (Cascading Style Sheets) → Untuk desain dan tata letak c) JavaScript → Untuk interaktivitas dan responsivitas B. Back-End (Server-Side): Bagian dari aplikasi yang menangani logika, penyimpanan data, dan komunikasi dengan database. Contohnya: PHP, Python, Node.js → Untuk pemrosesan data di server MySQL, PostgreSQL, MongoDB → Untuk menyimpan dan mengelola data API (Application Programming Interface) → Untuk komunikasi antar sistem C.
